📌 Requirements
Functional Requirements
1. Student registration and login
2. Skill entry and management
3. Job role selection
4. Skill gap analysis
5. Placement probability prediction
6. Course recommendations
7. Admin dashboard

Non-Functional Requirements
1. Secure authentication
2. Fast response time
3. User friendly interface
4. Cross browser compatibility
5. Secure data storage

Users
- Student
- Admin

Input
- Student name, CGPA, Skills,
  Job role, Internship details

Output
- Skill gap report
- Match percentage
- Placement probability %
- Course recommendations

👥 Users

Student
→ Register and login
→ Enter skills and CGPA
→ Select target job role
→ View skill gap report
→ View placement probability

Admin
→ Login to admin panel
→ Add and manage job roles
→ Add required skills per job
→ View all student reports
→ Monitor placement readiness

🧩 Modules

1. Authentication Module
A module that handles student registration, login and session management

2. Student Profile Module
A module that manages student skills, CGPA and internship details

3. Job Role Module
A module that handles job listings and required skills per job role

4. Skill Gap Analysis Module
A module that compares student skills vs job requirements and shows gap report

5. Placement Prediction Module
A module that uses ML to predict student placement probability
